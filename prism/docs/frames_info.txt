joints annotations:

after evaluating the joints location for
the top and side in all subdirectories, one can run
make_labels_forall.py for each image directory.
(e.g. for /ramdya-nas/SG/prism_data/191125_PR/Fly2/001_prism/behData/images)
this will generate a file named
joint_locations.csv
with all the joints locations. The structure of the file is:
	1st line: bodyparts, first top (first left then right) then side
	from 2nd line: img_name,x1,y2,score1,x2,y2,score2,...
			45 joints x 2 coordinates + 45 scores (1 per joint) = 135 entries



cropped frames are in two subdirectories of each "images" directory:

top_view_191125_PR_Fly1_001_prism
side_view_191125_PR_Fly1_001_prism
TOP: (440, bbox_width, 3)
SIDE: (260, bbox_width, 3)

annotations are saved in the following file:
top_view_191125_PR_Fly1_001_prismDeepCut_resnet50_jointTrackingDec13shuffle1_200000.csv
side_view_191125_PR_Fly1_001_prismDeepCut_resnet50_sideJointTrackingJan28shuffle1_200000.csv
structure:
	1st line: scorer (DeepCut_resnet50_jointTrackingDec13shuffle1_200000)
	2nd line: bodyparts, first left then right (body-coxa front L,coxa-femur front L,...)
	3rd line: coords, (x,y,likelihood,x,y,likelihood,...)
	from 4th line: img_name,x1,y1,score1,x2,y2,score2,...
			30 joints x 2 coordinates + 30 scores (1 per joint) = 90 entries
